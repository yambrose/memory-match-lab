# Memory Match Game

A basic memory card matching game built with JavaScript, HTML, and CSS.

### Features

- **Gameplay**: Flip two cards at a time to find matching pairs
- **Statistics**: Track moves, matches, streak, and elapsed time
- **Score Tracking**: Save your best scores with your name and date
- **High Scores Leaderboard**: View all saved scores ranked by number of moves
- **Responsive Design**: Works seamlessly on desktop and mobile devices

### How to Play

1. Flip two cards by clicking on them
2. If the cards match, they stay revealed and your match count increases
3. If they don't match, they flip back over
4. Continue until all pairs are matched
5. When you win, enter your name to save your score

### Game Statistics

- **Moves**: Total number of card flips
- **Matches**: Number of matched pairs found
- **Streak**: Current consecutive matches
- **Time**: Elapsed time in seconds
